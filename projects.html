Hybrid Point-Grid RadarNet

Academic Project, 2024 • Deep Learning Developer • Deep Learning, Autonomous Vehicles, Radar, Computer Vision

A novel hybrid neural network architecture for enhanced radar object detection in autonomous vehicles using the OpenPCDet framework.

Project details

Overview

Implemented a novel hybrid architecture that combines the strengths of both point-based and grid-based neural networks to improve object detection from radar data. By integrating point-wise feature extraction before grid rendering, the model achieves significantly better object orientation estimation. The architecture was replicated from a state-of-the-art research paper and validated on the VoD autonomous driving dataset, outperforming baseline methods.

Key features / highlights

  • Achieved up to 19.7% higher mean Average Precision (mAP) for vehicle detection.
  • Improved object orientation estimation through early point-wise feature extraction.
  • Utilized advanced point cloud processing techniques like GNNs and Kernel Point Convolutions (KPConv).
  • Successfully replicated and validated a research paper's architecture.

Tech & tools

  • OpenPCDet Framework
  • PyTorch
  • Graph Neural Networks (GNN)
  • Python
  • VoD Dataset
  • Point Cloud Processing

My role

I led the implementation of the hybrid architecture within the OpenPCDet framework. My responsibilities included adapting the codebase to integrate point-based and grid-based modules, training the model on the large-scale VoD dataset, and performing a rigorous validation to compare its performance against established baseline methods.

Development of Local and Global Path Planners for TurtleBot

Passion Project • Robotics Developer • Path Planning, Robotics, ROS, Algorithms

A complete navigation stack for a TurtleBot mobile robot, featuring custom global and local path planners for robust autonomous navigation.

Project details

Overview

This project involved the development and implementation of a full path planning system for a TurtleBot in a static environment. A global path planner was designed to compute the optimal route from a start to a goal position, ensuring it avoids known obstacles and minimizes travel distance. In parallel, a local path planner was created to operate in real-time, allowing the robot to adjust its path dynamically to navigate around unexpected obstacles and changes in the environment.

Key features / highlights

  • Developed a global path planner for optimal, long-range navigation.
  • Created a real-time local planner for dynamic obstacle avoidance.
  • Ensured seamless integration between the global and local planners.
  • Successfully demonstrated the system's effectiveness on a physical TurtleBot platform.

Tech & tools

  • ROS (Robot Operating System)
  • Gazebo
  • Python / C++
  • Path Planning Algorithms
  • TurtleBot
  • SLAM/Mapping

My role

I was responsible for designing and implementing both the global and local path planning algorithms from scratch. This included algorithm selection, software development within the ROS framework, and integration of the planners into a cohesive navigation stack. I also led the testing and validation phase, deploying the code on a real TurtleBot to demonstrate its effectiveness in a live environment.
